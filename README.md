# 浏览器插件：140字摘要

AI 生成网页一句话摘要（默认在 140-200 字之间），为您保留最重要的信息，不用花时间逐一点开阅览。

后台队列处理，侧边栏查看，历史记录可导出。支持一些永久免费模型。

## 功能特点

- **一键摘要**：点击工具栏图标打开侧边栏，自动对当前及后续打开的网页生成约 140 字中文摘要
- **后台队列**：文本加入队列前即提取，可安全关闭标签页，队列逐个处理不会丢失
- **倒金字塔摘要**：最重要的信息最先说出，关键数字必留，有效信息密度极高
- **免费模型**：可使用智谱 GLM-4-Flash / GLM-4.7-Flash 模型，官方提供永久免费使用
- **多模型支持**：还支持 Google Gemini、DeepSeek 等主流 AI 模型
- **所见即所送**：将浏览器中已完整渲染的页面转为 Markdown，AI 看到的就是用户看到的，如果你已经付费/登录，则处理的是付费墙内/已登录后的内容
- **历史记录**：最多 200 条，一键复制、导出文本文件、批量管理
- **跳过名单**：按页面或站点设置跳过，支持通配符规则
- **网站专属提示词**：为特定网站追加补充提示词，约束 AI 行为
- **自定义 Prompt**：可修改系统提示词，满足个性化摘要需求
- **设置备份**：JSON 格式导入/导出所有设置

### 手动安装

1. 下载最新 Release 中的 zip 包并解压
2. 打开 Chrome，访问 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」，选择解压后的文件夹

## 使用方法

1. 安装插件后，点击工具栏的插件图标打开侧边栏
2. 在设置页选择 AI 模型（推荐智谱 GLM-4-Flash，免费）并填入您自己的 API Key
3. 浏览网页时自动生成摘要，在侧边栏查看结果
4. 点击摘要区域一键复制，点击标题区域打开原文

## 权限说明

| 权限 | 用途 |
|------|------|
| `activeTab` | 获取当前活动标签页信息，触发摘要生成 |
| `scripting` | 在网页中注入脚本提取页面文本内容 |
| `storage` | 在浏览器本地存储设置、历史记录和缓存数据 |
| `sidePanel` | 在浏览器侧边栏中展示摘要界面 |
| `<all_urls>` | 允许在任意网页上提取文本生成摘要 |
