# 浏览器插件「140字摘要」的隐私政策

最后更新日期：2026年6月7日

## 1. 概述

「140字摘要」（以下简称"本扩展"）是由 Shuhang Li （James Li / lishuhang.me）开发的浏览器扩展程序。本隐私政策旨在向您说明本扩展如何收集、使用和保护您的信息。本扩展尊重您的隐私，并承诺不会收集任何不必要的个人信息。

## 2. 数据收集与存储

本扩展不收集、不存储、不分享任何用户个人身份信息。所有用户数据均存储在浏览器本地（chrome.storage.sync 和 chrome.storage.local），具体包括：

- **API Key**：存储在 chrome.storage.sync，仅用于向您选择的 AI 服务商发起请求，不会发送到任何第三方服务器
- **模型选择和系统提示词设置**：存储在 chrome.storage.sync
- **跳过名单**：存储在 chrome.storage.sync
- **网站专属提示词**：存储在 chrome.storage.sync
- **摘要历史记录**：存储在 chrome.storage.local，最多保留 200 条
- **URL 缓存**：存储在 chrome.storage.local，30分钟自动过期，最多 50 条

## 3. 数据使用

本扩展使用您的数据仅用于以下目的：向您选择的 AI 服务商发送网页文本以生成摘要。网页文本仅在您主动打开侧边栏时才会被提取和发送。AI 服务商的数据处理受其各自的隐私政策约束：

- 智谱开放平台：https://open.bigmodel.cn/
- Google AI Studio：https://makersuite.google.com/
- DeepSeek 平台：https://platform.deepseek.com/

## 4. 第三方服务

本扩展依赖以下第三方 AI 服务提供商提供的 API 服务：智谱开放平台、Google Gemini、DeepSeek。当您使用本扩展生成摘要时，网页文本将被发送到您选择的 AI 服务提供商的服务器进行处理。这些服务提供商有各自的隐私政策，您应当了解并同意其数据处理规则。本扩展本身不运营任何服务器，不中转或存储您的请求数据。

## 5. 权限说明

本扩展申请的浏览器权限及其用途如下：

| 权限 | 用途说明 |
|------|----------|
| activeTab | 获取当前活动标签页信息，用于触发摘要生成 |
| scripting | 在网页中注入脚本提取页面文本内容（含 Turndown 库用于 HTML→Markdown 转换） |
| storage | 在浏览器本地存储设置、历史记录和缓存数据 |
| sidePanel | 在浏览器侧边栏中展示摘要界面 |
| \<all_urls\> | 允许在任意网页上提取文本生成摘要 |

## 6. 数据安全

您的 API Key 和所有用户数据均存储在浏览器本地，不会被传输到本扩展的任何服务器。本扩展不包含任何第三方分析、跟踪或广告服务。您可以随时通过扩展设置中的"清空历史"功能删除所有本地存储的摘要记录和缓存，也可以通过卸载扩展彻底清除所有本地数据。

## 7. 儿童隐私

本扩展不面向 13 岁以下儿童，不会故意收集儿童个人信息。如果您发现本扩展无意中收集了儿童的信息，请联系我们，我们将立即删除相关数据。

## 8. 数据删除

您可以随时通过以下方式删除您的数据：在扩展设置中点击"清空"按钮可删除所有历史记录和缓存；卸载扩展后，浏览器将自动清除扩展的所有本地存储数据（包括 API Key、设置、历史记录等）。

## 9. 政策更新

我们可能会不定期更新本隐私政策。更新后的政策将在本页面发布，并更新页面顶部的"最后更新日期"。继续使用本扩展即表示您同意更新后的隐私政策。
